Feedendum: read, modify and write feeds easily
==============================================

**Feedendum** is a small library for
parsing, editing, sorting and producing Atom, RSS or RDF Feeds.

How to install
--------------

Simply get it from PyPi via ``pip``::

    python -m pip -U feedendum
